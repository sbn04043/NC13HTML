ML: Marked-Up Language
HTML: Hyper Text Makred-Up Language
ML은 <>로 설정할 값의 종류와 값이 표현된다.
ML은 <>로 값의 종류가 시작되고 </>로 끝나야 한다.
ex)	<dependency>
		<artifactId>org.projectlombok</artifactId>
		content
	</dependency>

추가로 설정 값을 넣을 수 있는데 attribute 이름과 값을 넣는다.
ex)	<dependency value="...">content</dependency>

태그 사이 요소가 없는 경우 <태그/>로 줄여 쓸 수 있다.
ex)	<br/>

HTML은 반드시 <HTML>로 시작하고 </HTML>로 종료된다.
HTML 파일을 저장할 때는 반드시 .html 확장자로 저장해야 한다.

HTML 안에는 Head와 Body가 있다.
head: 해당 html 문서의 설명과 
body: 웹 브라우저가 표현하는 부분
